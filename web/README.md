# FireURQ Web

This folder contains the TypeScript/web migration workspace.

Planned packages:

- `app` - React + Vite web player shell
- `packages/core` - URQL lexer/parser/IR/VM port target
- `packages/player` - compatibility player/runtime layer for rendering, assets, and FireURQ-style behavior

## Getting started

```bash
cd /Users/fireton/fieurq3/web
pnpm install
pnpm dev
```
